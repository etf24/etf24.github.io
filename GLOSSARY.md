# Glossary (Ubiquitous Language)

This glossary is intentionally minimal and only includes terms required by the current v1 feature scope.

| Deutsch               | English              | Definition                                                                                        |
|-----------------------|----------------------|---------------------------------------------------------------------------------------------------|
| Markenname            | Brand Name           | The product name is always written as `etf24` in lowercase.                                       |
| Portfolio             | Portfolio            | Sum of all positions in the user's holdings.                                                      |
| Asset                 | Asset                | A financial instrument (e.g., ETF, stock) used as a building block in the portfolio.              |
| Position              | Position             | The current holding of a specific asset in the portfolio.                                         |
| Depotwert (brutto)    | Market Value (gross) | Current value of a position as shown by the broker, before taxes and fees.                        |
| Nettoerlös            | Net Proceeds         | Value after taxes and fees if the position were sold.                                             |
| Investiertes Kapital  | Invested Capital     | Total amount the user has paid into a position (cost basis).                                      |
| Unrealisierter Gewinn | Unrealized Gain      | Gain or loss that exists on paper but has not been realized by a sale.                            |
| Realisierter Gewinn   | Realized Gain        | Gain or loss that has been realized by a sale.                                                    |
| Teilfreistellung      | Partial Exemption    | Tax-exempt portion of gains (e.g., 0%, 15%, 30%).                                                 |
| Steuerprofil          | Tax Profile          | Configured tax rates used for calculations (capital gains tax, solidarity surcharge, church tax). |
| Kapitalertragsteuer   | Capital Gains Tax    | Primary tax on realized capital gains.                                                            |
| Solidaritätszuschlag  | Solidarity Surcharge | Surcharge applied to the capital gains tax amount.                                                |
| Kirchensteuer         | Church Tax           | Surcharge applied to the capital gains tax amount (optional).                                     |
| Sparer-Pauschbetrag   | Saver Allowance      | Annual tax-free allowance on capital investment income, applied before capital gains taxation.    |
| Rebalancing           | Rebalancing          | Adjusting positions to reach target allocations.                                                  |
| Trade (Kauf/Verkauf)  | Trade (Buy/Sell)     | A purchase or sale executed as part of rebalancing.                                               |
| Gebühr                | Fee                  | Transaction cost applied to a trade (fixed and/or percentage).                                    |
| Sparrate              | Savings Rate         | Recurring cash contribution used for cash-only rebalancing.                                       |
| Rundung               | Rounding             | Rounding follows kaufmaennisches Runden (DIN 1333): 0-4 down, 5-9 up, on cent precision.          |
